API Demo — Backend en Vivo

No solo interfaz.

Este formulario es una demostración de un backend real. No envía un email a un servicio externo — valida los datos server-side con una API Route de Next.js, construye una respuesta estructurada y la devuelve al cliente.

El endpoint POST /api/contact verifica que todos los campos estén presentes, valida el formato del email con regex, sanitiza el mensaje y retorna un objeto JSON con el estado de la operación.

Esto demuestra que entiendo cómo conectar frontend y backend: el cliente no sabe nada de la validación — todo sucede en el servidor antes de responder.
